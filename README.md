# Medical Visit Manager

Medical Visit Manager is a web application backend for managing patients and their medical appointments.

The application contains two connected data entities:

- Patient
- Appointment

One patient can have many appointments. Each appointment belongs to one patient.

## Technologies

- Node.js
- Express.js
- JSON file storage

## Main features

- Create, read, update and delete patients
- Create, read, update and delete appointments
- List appointments for a selected patient
- Validate appointment time interval
- Prevent overlapping appointments
